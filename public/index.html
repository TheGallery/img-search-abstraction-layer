Image Search Abstraction Layer

This API is an abstraction of Imgur's API. You can search Imgur galleries and see your search history.
Check the endpoints and the examples below.

Endpoints:

https://floating-wildwood-37090.herokuapp.com/api/search/{query}

Returns an array with image details based on the query provided.

Parameters

{query}: The search term.

Query Parameters

[?offset=n] (optonal): You can pass this parameter to get the n-th result page.

Example usage:

https://floating-wildwood-37090.herokuapp.com/api/search/dogs?offset=3

Example output:

[{"url":"http://i.imgur.com/kS0pAwr.png","description":"At least some Police agencies know how to interact with dogs.","context":"https://i.imgur.com/kS0pAwr"},{"url":"http://i.imgur.com/Bd4EFsp.gif","description":"Dogs have invented cookie-powered teleportation. We haven't even figured out renewable energy yet.","context":"https://i.imgur.com/Bd4EFsp"}, ...]

https://floating-wildwood-37090.herokuapp.com/api/history

Example usage:

https://floating-wildwood-37090.herokuapp.com/api/history
